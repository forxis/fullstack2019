When you open up browser it sends request to server → HTTP GET https://fullstack-exampleapp.herokuapp.com/spa
↓
While loading HTML browser requests CSS file as well so it sends another request to server → HTTP GET https://fullstack-exampleapp.herokuapp.com/main.css
↓
After loading CSS file browser sends another request to server to load JS → HTTP GET https://fullstack-exampleapp.herokuapp.com/spa.js
↓
When a user leaves a note on the browser, browser starts executing JS code → and sends a HTTP POST 201 request https://fullstack-exampleapp.herokuapp.com/new_note_spa to add an additional value to data.json
↓
Once the new note is added JS script fires again to display the refreshed notes
