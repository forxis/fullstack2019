When you open up browser it sends request to server → HTTP GET https://fullstack-exampleapp.herokuapp.com/notes
↓
While loading HTML browser requests CSS file as well so it sends another request to server → HTTP GET https://fullstack-exampleapp.herokuapp.com/main.css
↓
After loading CSS file browser sends another request to server to load JS → HTTP GET https://fullstack-exampleapp.herokuapp.com/main.js
↓
When a user leaves a note on the browser, browser starts executing JS code
↓
Browser requests JSON data from server by sending request → HTTP GET https://fullstack-exampleapp.herokuapp.com/data.json
↓
Server sends to browser the content of the data.json file with the added value → [{ content: "HTML is easy", date: "2019-05-23" }, ...]
↓
Browser loads event handler function that displays all the notes values from data.json
